# 文制星设置界面设计规范



## 一、整体布局结构

| 参数 | 值 |
|------|-----|
| **布局方式** | 左右分栏布局 |
| **总高度** | h-screen |
| **左侧面板** | w-[280px] 固定宽度 |
| **右侧内容区** | flex-1 自适应宽度 |

**结构层级**：
```
Settings.vue (根容器)
  ├── 左侧导航栏 (w-[280px])
  │     ├── 标题区 (px-6 pt-5 pb-5)
  │     └── 导航按钮组 (flex-1 px-4 pb-4)
  └── 右侧内容区 (flex-1)
        ├── 顶部标题栏 (px-8 py-5)
        └── 内容滚动区 (flex-1 p-8)
```

## 九、交互特性

1. **侧边栏指示器动画**：点击导航按钮时，彩色指示器平滑移动到对应位置
2. **分类筛选指示器**：模板分类切换时，白色指示器平滑滑动
3. **开关动画**：Toggle 开关使用弹性缓动曲线 (cubic-bezier)
4. **选中标识过渡**：模板选中状态使用 Transition 组件实现淡入淡出
5. **悬停提示**：操作按钮悬停显示 Tooltip
6. **删除确认**：自定义模板删除前弹出确认对话框
